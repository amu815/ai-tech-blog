## Ollamaの特徴
Ollamaは、以下のような特徴を持っています。
* 無料で使い放題：Ollamaは完全無料で使うことができます。
* ローカル環境での動作：Ollamaはクラウドではなくローカル環境で動作します。
* 高度な言語理解能力：Ollamaは、高度な言語理解能力を持ちます。

## Ollamaの使い方
Ollamaの使い方は、以下の通りです。
```bash

pip install ollama
```
